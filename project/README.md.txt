# Xbox Releases Page

This is a simple and responsive webpage that lists new Xbox game releases for April 2025. It includes a dynamic navigation bar, multiple content sections, and a working comment system.

## Features

- ✅ Dynamic navigation bar built using JavaScript
- ✅ Four main sections with headings and paragraphs
- ✅ Scrolls smoothly to sections on nav click
- ✅ Highlights the active section as you scroll
- ✅ Fully functional comment form:
  - Requires name, email (with `@`), and comment
  - Displays submitted comments on the page
- ✅ Responsive design for desktop, tablet, and mobile

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Use

1. Open `index.html` in your browser.
2. Scroll through the sections or use the navigation bar.
3. Add a comment at the bottom and see it appear instantly.

## Skills Demonstrated

- DOM manipulation

- Form validation
- Responsive design
